出国后，网易云音乐怎么办

2019-07-20

也是在云村呆过好几年的人，这两年云村版权不济，像「周董」这样的门面担当的版权全在企鹅手上。不过云村仍然有我非常喜欢的小众歌单，这是我离不开它的原因。

到新加坡的第二天，照常打开电脑，戴上耳机，打开云村。突然发现在国内还正常播放的歌单被 ban 了一大半 ... 点击播放提示：「由于版权保护，您所在的地区暂时无法使用。」这句话听人说起过，当时并未在意。

这时候一般有两种解决方案：「科学上网」和「换音乐软件」。对，不然用代理，让你假装还在国内；不然换一个软件，并且想办法把喜欢的歌都迁移过去。本着「入乡随俗」的原则（大概是在剑桥公社时，黑人小哥也会用微信加我好友一样的意思），我选择了第二种方案。用本土化（国际化）的软件，能够减少很多麻烦。

导入歌单

歌单导出在网站TuneMyMusic，这个网站打通了「Spotify」、「Apple Music」等近乎国外所有的主流音乐媒体平台。

打开网站，点击「上传文件」，将刚导出的「TXT」文件上传即可。接下来按照你的意愿，选择导入的平台。

其他平台迁移思路

整个过程分为导出和导入，导入没啥好说的。导出对不同平台给这些建议：

  • QQ 音乐：腾讯导入歌单做得简单无比，甚至你直接把其他平台的「整个网页」复制下来[2]这样暴力操作，他也能帮你把歌单解析出来。但是导出显得比较困难，可能只能用 Python 自己爬，请参考：Python抓取QQ音乐歌单并分析
  • 虾米音乐：虾米也可以写 Python 自己爬，或者尝试用 Chrome 解析网页源代码，Excel 里再用查找替代将歌名过滤出来
  • Spotify：Spotify 有自己的导出歌单功能

不过以上方法的复杂度 ... 是不是不如一首歌一首歌找回来。
