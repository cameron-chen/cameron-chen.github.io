如何开始一场直播

2019-07-27

很偶然的机会，我需要接触到直播，并且还不是那种打开手机 App 就能够进行的形式。我需要裁切画面，邀请其他嘉宾（通过视频），布置弹幕。可能是需求比较「小众」的原因，有时候有种翻遍 Google 也找不到解决方案的感觉。

所以将整个过程记录下来。如果只是简单的开播，你只需要看前面一小部分内容就能开始。而后面的几个环节，有需求再看就好。

直播软件

直播软件分为三类：「官方桌面软件」、「官方移动端软件」和「OBS」。

三者的区别是，移动端软件操作很简单，根据官方的提示就能直播，而直播形式通常也很简单，就是传输摄像头的画面；官方桌面软件根据官方教程也不难，这种选择通常能够传递更复杂的画面，如：「电脑桌面」、「网页」、「摄像头画面」；OBS 软件本身不是专为直播服务的，其能够完成「视频录制」、「视频推流」等自由度更高的工作。

  • 官方软件：「B 站直播姬」、「虎牙直播客户端」等，这些软件一般都有桌面和移动端两个版本
  • OBS：推荐使用 「OBS Studio」

接下来就以 OBS Studio 为核心软件，讲解如何开播。

实现环境

  • 软件：OBS Studio 23.0.2（64 bit）、Skype Win10 应用
  • 插件：NewTek NDI 3.8 Runtime

基本使用

基本使用「吃」基本教程就好，看完下面这个教程开播就没问题了。但这个教程是「游戏主播」导向，所以如果有其他需求，比如：举办一场线上访谈 ... 请继续看后面的内容。提示：后面小节的内容请了解基本操作和术语之后继续阅读。

基础教程推荐：B 站 | OBS 直播教程

连麦

连麦这个需求出现于你希望让其他主播来你这「串串门」，或者你邀请了嘉宾在做一场「访谈直播」。B 站直播姬有类似叫「PK 」的功能，但是画面限制的很死，就是左边你的画面，右边另一个主播的画面。

OBS 上用 Skype 的 NDI 串流[1]功能，即可将你的音视频及你通话对象的音视频传递至 OBS。准备工作如下：

  1. 下载 OBS 插件并放置于指定位置
  2. 安装插件支持：NDI Runtime
  3. Skype 允许使用 NDI 功能
  4. OBS 上添加 NDI 源

其上步骤1, 2可以简化为安装「一键安装包」，地址：obs-ndi，下载obs-ndi-x.x.0-Windows-Installer.exe。版本号可能不同，下载最新版就好。

下载一键安装包 ▼

如果不能安装一键包，请跟着下面的步骤手动安装。（我安装的时候网络抽风没能装好，如果你正好也不能装不要惊讶）

OBS 插件下载，并添加至插件文件夹

下载插件包，也是同样的地址：obs-ndi，下载obs-ndi-x.x.0-Windows.zip。

解压之后有：data和obs-plugins两个文件夹，将这两个文件夹复制到 OBS 的安装文件夹，此文件夹应该有uninstall.exe程序。

插件复制地址 ▼

安装插件支持

插件还需要单独的程序支持，一个叫「NewTek NDI Runtime」的软件。

  • 官方下载地址：NewTek NDI Runtime
  • 官网抽风时的备用地址：云盘 | NewTek NDI 3.8 Runtime，密码：qqjp

Skype 开启 NDI 传输

打开「设置」，点击「通话」，「高级」，开启「允许使用 NDI」。

开启 Skype NDI 传输 ▼

OBS 添加 NDI 源

在 OBS 源中，选择 NDI 源。注：在视频通话之后 Skype 的源才会显示出来。

OBS 中添加 NDI 源 ▼

弹幕

弹幕有两种实现方式，一种直接使用软件，它会自动生成源，可以直接添加至直播画面中；另一种是网页生成弹幕，OBS 再捕捉网页源。两种方式都有对应的服务提供商。

  • 软件推荐：弹幕库
  • 网页源推荐：智播云插件，登陆并选择插件，添加网页源即可使用

PPT 广播

这是一个更加小众的部分。你可以理解为：你邀请了一名嘉宾，但是他需要展示 PPT，而你在用 OBS 软件将展示过程推送出来。这样的场景下，需要用到「PPT 广播」[2]这个功能。想必你已经猜到，将「PPT 广播」生成的链接，添加到「网页源」，即可在 OBS 上看到 PPT 画面，而嘉宾可以远程控制。

  • 软件推荐：Presentain

  1. NDI 串流：允许截取 Skype 的视频通话，并且传输给 OBS 使用。技术详情见：NDI网络设备接口技术简介 ↩︎

  2. PPT 广播：将 PPT 画面远程展示出来，收看者通过链接（浏览器）查看展示内容 ↩︎
